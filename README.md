# Student Record Management System (CSV-Based)

## 📌 Overview

This is a simple Python project that manages student records using a CSV file. It allows users to add student details, calculate marks, assign grades, and view stored data.

## 🚀 Features

* Add new student records
* Automatically calculate total and average
* Assign grades based on performance
* Store data in a CSV file
* View all student records

## 🛠️ Technologies Used

* Python
* CSV module
* OS module

## 📂 File Structure

* `students.csv` → Stores student data
* `main.py` → Main program file

## ▶️ How to Run

1. Make sure Python is installed
2. Save the code in a `.py` file
3. Run the program:

   ```bash
   python main.py
   ```

## 📊 Grading Criteria

* A → Average ≥ 90
* B → Average ≥ 75
* C → Average ≥ 50
* Fail → Average < 50
